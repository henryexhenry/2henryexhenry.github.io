Posted on 2019-09-19

Flask is a microframwork to help user build a website. It depends on the Jinja template engine and the Werkzeug WSGI toolkit.

Start a flask project

  • Installation

    • pip install Flask

  • Create a python file, eg.(temp.py)

  • configuration

    • export FLASK_APP=temp.py

    To tell the flask engine which file should be run when typing in flask run

    • export FLASK_ENV=development

    Switch flask into debug mode

  • start running the server

    • flask run

    Type this command everytime you want to run your server.

    Now the server is running, try typing in http://127.0.0.1:5000/ in your browser.
