Installations

  • Git
    • What is Git?
      • Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
    • Download
  • Node.js
    • What is Node.js
      • As an asynchronous event driven JavaScript runtime, Node.js is designed to build scalable network applications.
    • Download
    • win10: install nodejs in C: to advoid bugs
  • Hexo
    • What is Hexo
      • Hexo is a fast, simple and powerful blog framework. Hexo parses your article by Markdown rendering engin, it can produce static files with beautiful theme in a few second.
    • install Hexo by npm(node package management) from node.js
      • npm install -g hexo-cli
  • hexo-deployer-git
    • this is a plugin for deploying hexo blog on github
    • npm install hexo-deployer-git --save

Steps

  • initialise a Github repository

  • login github locally

    • open .ssh folder, copy the ssh key string by cat id_rsa.pub

    • open github webpage - settings - SSH keys - add ssh keys

    • paste the ssh key here

  • build blog server

    • initialization
      • initialise a new blog
      • by hexo init blog
    • generation
      • generate the blog with modifications
      • by hexo generate or hexo g
    • run
      • run the blog server locally for preview
      • by hexo server or hexo s
    • deploy the blog
      • by hexo deploy or hexo d

  • configuration

    • deploy blog server on Github

      modify the configuration for the hexo deploy command

      • open _config.yml file

      • modify

    • restart

      Restart the server to update the configurations

      • hexo clean
      • hexo g
      • hexo d

      Now the Hexo blog is running on Github server.

      Blog url: <username>.gitbub.io
